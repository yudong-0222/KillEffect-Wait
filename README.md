A simple kill effect plugin.    
Works on 1.8 server!

## Commands
![img.png](img.png)

+ killeffect gui: Open a gui for select the kill effect
+ killeffect remove: Remove an effect which player is using.
+ killeffect respawn: Respawn immediately. **(NEED PERM)**
